# BritNgo - A Fictional Charity Website Built with Python / Django

Description:

BritNGO is a fictional charity website built to publish articles and also generate funds to support education for kids around the world

Technologies:
HTML
CSS
JavaScript
Stripe
Python
Django

Features:
Functioning contact form
Donation page
Multiple paged website - Home page, Blog / Blog Detail Page, Contact Page, Donation Page and AboutUs Page
